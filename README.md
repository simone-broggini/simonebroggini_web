# Simone Broggini - Personal Website

This is the source code for my personal website, available at [simone-broggini.github.io/simonebroggini_web/](https://simone-broggini.github.io/simonebroggini_web/).

## Description

A clean, responsive, and professional portfolio website built with HTML and CSS. The site showcases my work, provides information about me, and includes ways to get in touch.

## Running Locally

To run the website on your local machine, navigate to the project's `public` directory and use Python's built-in HTTP server:

```bash
cd public
python3 -m http.server
```

The website will be available at `http://localhost:8000`.
